Here are listed some Open Source projects which can be used as a starting point for developing with the {{ site.data.strings["vaadin_plugin"].title }}:

### DS Vaadin Beverage Buddy

<img class="screenshot" src="../assets/images/beverage_buddy.png" /> 

Single module Vaadin Flow application with the Gradle Flow plugin. 

The sources can be found in the [Github Example Repository](https://github.com/devsoap/examples/tree/master/beverage-starter-flow-gradle).

### DS Vaadin Bookstore Starter

<img class="screenshot" src="../assets/images/bookstore.png" /> 

Multi-module Vaadin Flow Starter with the Gradle Flow plugin.

The sources can be found in the [Github Example Repository](https://github.com/devsoap/examples/tree/master/bookstore-starter-flow-gradle).

#### Kotlin + Karibu DSL

<img class="screenshot" src="../assets/images/karibu.png" /> 

Single module Vaadin Flow application written in Kotlin using the Gradle Flow plugin.

The sources can be found in the [Github Example Repository](https://github.com/devsoap/examples/tree/master/karibu10-helloworld-application).
